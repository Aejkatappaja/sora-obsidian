# Sora for Obsidian

A dark theme for [Obsidian](https://obsidian.md). Ethereal cyan, cool silver, deep OLED blacks.

Part of the [Sora](https://github.com/Aejkatappaja/sora) colorscheme family.

![screenshot](screenshot.jpg)

## Install

### From community themes (coming soon)

Settings > Appearance > Themes > Search "Sora"

### Manual

Copy the theme folder to your vault:

```bash
cp -r extras/obsidian/ <vault>/.obsidian/themes/Sora/
```

Then go to Settings > Appearance > Themes > Sora.
